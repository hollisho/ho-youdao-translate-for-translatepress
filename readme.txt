为TranslatePress添加有道翻译API支持，实现自动化翻译功能。

== Description ==

Ho YouDao Translate For TranslatePress 是一个TranslatePress插件的扩展，它为TranslatePress添加了有道翻译API支持。通过这个插件，您可以使用有道翻译的API来自动翻译您的WordPress网站内容。

主要特性：

* 支持有道翻译API V2版本
* 支持自定义词典（可选）
* 简单的配置界面
* 支持多语言翻译
* 与TranslatePress完美集成

= 要求 =

* WordPress 6.0或更高版本
* PHP 7.2或更高版本
* TranslatePress插件

== Installation ==

1. 上传插件到`/wp-content/plugins/`目录，或通过WordPress插件页面直接安装
2. 在WordPress后台启用插件
3. 转到TranslatePress > 设置 > 自动翻译页面
4. 选择"有道翻译"作为翻译引擎
5. 输入您的有道翻译API密钥（格式：应用程序ID#应用程序密钥#VOCABID（可选））
6. 保存设置

== Frequently Asked Questions ==

= 如何获取有道翻译API密钥？ =

1. 访问[有道智云](https://ai.youdao.com/)
2. 注册/登录账号
3. 创建自然语言翻译服务
4. 获取应用ID和应用密钥

= 支持哪些语言？ =

插件支持有道翻译API支持的所有语言，包括但不限于：
* 中文（简体）
* 中文（繁体）
* 英语
* 日语
* 韩语
* 法语
* 西班牙语
等

= 翻译有字数限制吗？ =

有道翻译API有调用频率和字数限制，具体限制取决于您的API套餐类型。请参考有道智云官方文档了解详情。
